# Build a Job Application Form

You will build an online job application form.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should have a <code>div</code> element with the class <code>container</code>.

2. Inside the <code>div</code> element, you should have a <code>form</code> element.

3. The form should contain an <code>input</code> element with the type <code>text</code> and the id <code>name</code> for entering the user's full name.

4. You should have another <code>input</code> element with the type <code>email</code> and the id <code>email</code> for entering the user's email address.

5. The form should include a <code>select</code> element with the id <code>position</code> that allows users to select a job position.

6. You should have a <code>fieldset</code> element with class of <code>radio-group</code>.

7. Inside <code>.radio-group</code> you should have a set of <code>input</code> elements with the type <code>radio</code> and relevant labels for selecting availability options (e.g., Full-Time, Part-Time). The group name should be <code>availability</code>.

8. You should have a <code>textarea</code> element with the id <code>message</code> for entering a message.

9. You should associate every <code>input</code> element with a <code>label</code> element.

10. You should have a <code>button</code> element with the type <code>submit</code> for submitting the form.

11. Add a <code>:focus</code> pseudo-class to the <code>input</code> and <code>textarea</code> elements to change their border color when focused.

12. The <code>input</code>, <code>select</code> and <code>textarea</code> elements should have an <code>:invalid</code> pseudo-class that changes the border color to red when invalid input is detected.

13. The <code>input</code>, <code>select</code> and <code>textarea</code> elements should have a <code>:valid</code> pseudo-class that changes the border color to green when valid input is entered.

14. The <code>button</code> element should have a <code>:hover</code> pseudo-class that changes the background color when hovered over.

15. Use the <code>:checked</code> pseudo-class on <code>.radio-group input[type="radio"]</code> to add a border color, background color and a box shadow when the radio button is selected.

16. Use the <code>:checked</code> pseudo-class on radio buttons to change the text color of the associated label when the option is selected.

17. Add a <code>:first-of-type</code> pseudo-class to the <code>input</code> element to style the first input field differently. (e.g., rounded corners).

**Note:** Be sure to link your stylesheet in your HTML and apply your CSS.
