# Build a Book Catalog Table

You will build a book catalog app that displays various books and their information.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should create a <code>table</code> element that lists book information.

2. Your <code>table</code> should have a table head element with one row in it.

3. The row in your table head element should have four table header elements, with the text of <code>Title</code>, <code>Author</code>, <code>Genre</code>, and <code>Publication Year</code>, in that order.

4. Your <code>table</code> should have a table body element with at least five rows in it.

5. Each row in your table body should have four table data elements that display the book's Title, Author, Genre, and Publication Year.

6. Your <code>table</code> should have a table footer element with one row in it.

7. The row in your table footer element should have a table data element that spans four columns and has the text <code>Total Books: N</code> where <code>N</code> should be replaced by the number of books in your table.
