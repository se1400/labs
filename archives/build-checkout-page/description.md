# Build a Checkout Page

You will build out a check out page for an e-commerce solution.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should have an <code>h1</code> element with the text <code>Checkout</code>.

2. You should have two <code>section</code> elements immediately after the <code>h1</code> element.

3. You should have an <code>h2</code> element with the text <code>Your Cart</code> within the first <code>section</code>.

4. You should have an image of an item in the first <code>section</code> with appropriate alternate text. You can use this image if you would like: <code>https://se1400.github.io/labs/assets/cube.jpg</code>

5. You should have an <code>h2</code> element with the text <code>Payment Information</code> within the second <code>section</code>.

6. You should have a <code>form</code> element within the second <code>section</code>.

7. You should have an <code>input</code> with an <code>id</code> and <code>name</code> of <code>card-name</code>, and a <code>type</code> of <code>text</code> within your <code>form</code> and a <code>label</code> associated with it.

8. You should have an <code>input</code> with an <code>id</code> and <code>name</code> of <code>card-number</code>, and a <code>type</code> of <code>text</code> within your <code>form</code> and a <code>label</code> associated with it.

9. You should have at least two <code>input</code> elements with the <code>required</code> attribute.

10. You should include a <code>span</code> element with the text <code>*</code> and <code>aria-hidden</code> set to <code>true</code> inside the <code>label</code> element for each required input, so that required fields are visually indicated.

11. You should have a <code>p</code> element with a help text that explains the required card number format, placed immediately after the card number input. The <code>p</code> should have an <code>id</code> of <code>card-number-help</code> and be referenced by the card number input using <code>aria-describedby</code>.
