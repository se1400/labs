# Structure an HTML Document

In this lab, you'll practice structuring an HTML document using semantic elements. You will build a Utah Tech University page that demonstrates proper HTML5 document structure.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

### Document Foundation

1. Your document should start with a <code>&lt;!doctype html&gt;</code> declaration.

2. You should have an <code>&lt;html&gt;</code> element with a <code>lang</code> attribute set to <code>en</code> that wraps all your content.

3. Inside the <code>&lt;html&gt;</code> element, you should have a <code>&lt;head&gt;</code> element followed by a <code>&lt;body&gt;</code> element.

4. Your <code>&lt;head&gt;</code> element should contain a <code>&lt;meta&gt;</code> element with a <code>charset</code> attribute set to <code>utf-8</code>.

5. Your <code>&lt;head&gt;</code> element should contain a <code>&lt;meta&gt;</code> element with a <code>name</code> attribute set to <code>viewport</code> and a <code>content</code> attribute set to <code>width=device-width, initial-scale=1.0</code>.

6. Your <code>&lt;head&gt;</code> element should contain a <code>&lt;title&gt;</code> element with the text <code>Utah Tech University</code>.

### Header Section

7. Inside your <code>&lt;body&gt;</code>, you should have a <code>&lt;header&gt;</code> element as the first child.

8. Your <code>&lt;header&gt;</code> element should contain an <code>&lt;h1&gt;</code> element with the text <code>Utah Tech University</code>.

9. After the <code>&lt;h1&gt;</code>, your <code>&lt;header&gt;</code> should contain a <code>&lt;p&gt;</code> element with the text <code>Active Learning. Active Life.</code>

### Navigation Section

10. After your <code>&lt;header&gt;</code>, you should have a <code>&lt;nav&gt;</code> element containing the text <code>Home | Admissions | Academics | Campus Life</code>.

### Main Content

11. After your <code>&lt;nav&gt;</code>, you should have a <code>&lt;main&gt;</code> element.

12. Inside your <code>&lt;main&gt;</code> element, you should have a <code>&lt;section&gt;</code> element as the first child.

13. Your first <code>&lt;section&gt;</code> should contain an <code>&lt;h2&gt;</code> element with the text <code>Welcome to Utah Tech</code>.

14. After the <code>&lt;h2&gt;</code>, add a <code>&lt;p&gt;</code> element with the text <code>Utah Tech University is located in St. George, Utah. Our campus sits at the base of beautiful red rock cliffs in Southern Utah.</code>

15. Add another <code>&lt;p&gt;</code> element with the text <code>We are committed to active learning and helping students succeed.</code>

16. After the first <code>&lt;section&gt;</code>, add a second <code>&lt;section&gt;</code> element inside <code>&lt;main&gt;</code>.

17. Your second <code>&lt;section&gt;</code> should contain an <code>&lt;h3&gt;</code> element with the text <code>Why Choose Utah Tech?</code>

18. After the <code>&lt;h3&gt;</code>, add a <code>&lt;p&gt;</code> element with the text <code>Small class sizes with hands-on learning.</code>

19. Add another <code>&lt;p&gt;</code> element with the text <code>Over 200 programs to choose from.</code>

20. Add another <code>&lt;p&gt;</code> element with the text <code>Beautiful weather and outdoor adventures.</code>

21. After the second <code>&lt;section&gt;</code>, add an <code>&lt;aside&gt;</code> element inside <code>&lt;main&gt;</code>.

22. Your <code>&lt;aside&gt;</code> should contain an <code>&lt;h4&gt;</code> element with the text <code>Visit Campus</code>.

23. After the <code>&lt;h4&gt;</code>, add a <code>&lt;p&gt;</code> element containing the text <code>Schedule a tour today!</code> followed by a <code>&lt;br&gt;</code> element, then the text <code>Call us at (435) 652-7500</code>.

### Footer Section

24. After your <code>&lt;main&gt;</code>, you should have a <code>&lt;footer&gt;</code> element.

25. Your <code>&lt;footer&gt;</code> should start with an <code>&lt;hr&gt;</code> element.

26. After the <code>&lt;hr&gt;</code>, add a <code>&lt;p&gt;</code> element with the text <code>Copyright 2027 Utah Tech University</code>.

27. After the <code>&lt;p&gt;</code>, add an <code>&lt;address&gt;</code> element containing the text <code>225 S 700 E</code> followed by a <code>&lt;br&gt;</code> element, then the text <code>St. George, UT 84770</code>.

**Note:** This lab focuses on HTML structure only. No CSS is required.
