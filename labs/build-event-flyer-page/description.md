# Build an Event Flyer Page

You will build an online event flyer.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should have a <code>header</code> element within the <code>body</code>.

2. Your <code>header</code> element should have an image in it for your event, and an <code>h1</code> in it with your event title, in that order. You can use this image if you would like: <code>https://cdn.freecodecamp.org/curriculum/labs/event.jpg</code>

3. You should have a <code>main</code> element within the <code>body</code>.

4. Your <code>main</code> element should have at least two <code>section</code> elements within it showcasing the event features.

5. Your <code>section</code> elements each should have an <code>h2</code> within them.

6. Your <code>body</code> should have a top and bottom padding of <code>50px</code>.

7. Your <code>body</code> should have a top and bottom margin of <code>0</code>, and a left and right margin that centers itself.

8. Your <code>body</code> should have a width set relative to the width of the viewport.

9. Your <code>body</code> should use the <code>calc</code> function to set its <code>min-height</code> property to 100% of the viewport's height minus all padding applied to the top and bottom of the body.

10. You should have at least one <code>hr</code> within your flyer.

11. You should set the width of all <code>hr</code> and <code>section</code> elements to a percent value relative to its parent.

**Note:** Be sure to link your stylesheet in your HTML and apply your CSS.
