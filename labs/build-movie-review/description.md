# Build a Movie Review

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should have a <code>main</code> element.

2. Inside the <code>main</code> element, you should have an <code>h1</code> element for the movie title.

3. Below the <code>h1</code> element, you should have an <code>img</code> element displaying the movie cover. Your <code>img</code> element should have a descriptive <code>alt</code> text describing the image. You are free to use the following image if you like: <code>https://se1400.github.io/labs/assets/rise-beyond-2.webp</code>.

4. You should have a <code>p</code> element containing a brief movie description.

5. You should have another <code>p</code> element to display the movie rating. Within it, you should have these items in the listed order:
   - A <code>strong</code> element with the text <code>Movie Rating:</code>.
   - A <code>span</code> element with an <code>aria-hidden</code> attribute set to <code>true</code> containing a visual representation of the rating using stars <code>⭐⭐⭐⭐⭐⭐⭐⭐⭐☆</code>.
   - A numerical value, representing the rating, in parentheses (e.g. <code>9.2/10</code>) after the span.

6. You should have an <code>h2</code> element with the text <code>Cast Members</code>.

7. You should have a <code>ul</code> element.

8. Inside the <code>ul</code> element, you should have multiple <code>li</code> elements each containing a <code>strong</code> element for the actor's name followed by the corresponding character name preceded by the text <code>as</code>. (e.g., <code>James Holloway as Ethan Carter</code>).
