# Design a Business Card

You will build a digital business card.

**Objective:** Fulfill the user stories below and get all the tests to pass to complete the lab.

## User Stories:

1. You should have a <code>div</code> with a <code>class</code> attribute with a value of <code>business-card</code> that will contain all the other elements.

2. Inside the <code>.business-card</code> element, there should be an <code>img</code> element with a <code>class</code> attribute with a value of <code>profile-image</code>. You can set the image source to <code>https://se1400.github.io/labs/assets/flower.jpg</code> if you like. There should be an <code>alt</code> with a meaningful description.

3. Inside the <code>.business-card</code> element, after the <code>img</code> element, you should have three <code>p</code> elements with a <code>class</code> attribute that has, respectively, a value of <code>full-name</code>, <code>designation</code>, and <code>company</code>.

4. The first <code>p</code> element should contain your name.

5. The second <code>p</code> element should contain your designation.

6. The third <code>p</code> element should have your company name.

7. There should be an <code>hr</code> element below the third <code>p</code> element.

8. After the <code>hr</code> element, there should be two <code>p</code> elements. In the first <code>p</code> element, you should have an email address as the text. In the second <code>p</code> element, the text should be a phone number.

9. After the two <code>p</code> elements, there should be an <code>a</code> element with a class of <code>portfolio-link</code>, the text <code>Portfolio</code>, and it should link to a valid URL.

10. There should be an <code>hr</code> element after the <code>a</code> element containing the portfolio link.

11. You should have another <code>div</code> element with a <code>class</code> attribute with a value of <code>social-media</code>. Within this element, there should be an <code>h2</code> element with the text <code>Connect with me</code>.

12. Inside the <code>.social-media</code> elements, there should be three <code>a</code> elements. The <code>a</code> elements should have the text <code>Twitter</code>, <code>LinkedIn</code> and <code>GitHub</code> respectively with links to valid websites.

13. Your page background should be <code>rosybrown</code>. The overall font should be <code>Arial</code>, with a fallback of <code>sans-serif</code>.

14. The <code>.business-card</code> selector should have properties to set the element as <code>300px</code> wide and a background color of your choice. Also you should set a <code>padding</code> of <code>20px</code> all around and a top margin of <code>100px</code>. The text should be center aligned and the font size should be <code>16px</code>. The left and right margins should be set to <code>auto</code>.

15. The <code>.profile-image</code> selector should have a <code>max-width</code> property with a value of <code>100%</code>.

16. All of the <code>p</code> elements should have a top and bottom margin of <code>5px</code>.

17. All links on the page should not be underlined.
